# Task Ninja 2.0

Task Ninja 2.0 is an app that allows you to manage tasks efficiently.

## Features
- Add tasks based on priority and progress status
- Update tasks
- Delete tasks
- See a vertical list of tasks with a proper Kanban dashboard
